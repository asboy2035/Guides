**Astronomy** (or should we say *Ashtronomy*) is a modern web page that lets you explore all the planets of our Solar System with a clean, interactive UI.

[<button class="borderedButton">Visit</button>](https://ashtronomy.pages.dev/)

---

## Features

- #### Explore the Solar System
  Instantly learn key facts about Mercury, Venus, Earth, Mars, Jupiter, Saturn, Uranus, and Neptune.
- #### Interactive Planet Selector
  Quickly switch between planets with a beautiful floating sidebar.
- #### Stunning Visuals
  Gorgeous 3D-style planet renders bring the experience to life. ✨
- #### Key Planet Information
  View important facts like:
  - Position from the Sun
  - Average Temperature
  - Number of Moons
  - Distance from the Sun

---

## Technical Details

- Built with: **HTML, CSS, JavaScript**
- 3D planet visualizations powered by web technologies.
- Fully responsive — works beautifully on desktop and mobile.
- Static site, super lightweight and fast.

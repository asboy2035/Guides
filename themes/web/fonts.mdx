<Note>Link to the stylesheet using the CDN URL for the fonts below, then use the font name in your CSS.</Note>

## Example Implementation
Link to stylesheet in your HTML or CSS:

<CodeGroup>
  ```html icon="code" HTML
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/asboy2035/fonts@master/css/DTGetaiGroteskDisplayBlack.css" type="text/css" />
  ```
  ```css icon="paintbrush" CSS
  @import url("https://cdn.jsdelivr.net/gh/asboy2035/fonts@master/css/DTGetaiGroteskDisplayBlack.css");
  ```
</CodeGroup>

Then, use the font in CSS:
```css icon="paint-roller" focus={2} CSS Code
body {
  font-family: "DT Getai Grotesk Display Black", monospace
}
```

## Transparency

  ```
  https://cdn.jsdelivr.net/gh/asboy2035/Discord-Mods@main/Transparency.css
  ```
  This theme provides transparency to the Discord UI. Also, be sure to check out Demureify and Semi-transparent main that were designed to improve this theme.
  > Note: You can get translucency on macOS with [Vencord](https://vencord.dev).

<Frame>
  ![Transparency](https://cdn.jsdelivr.net/gh/asboy2035/Discord-Mods@main/screenshots/transparency.png)
</Frame>

  https://cdn.jsdelivr.net/gh/asboy2035/Discord-Mods@main/DemureifyNonTransparent.css

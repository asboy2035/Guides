Note: You will need a Discord client mod to use these styles. Check out [Vencord](https://vencord.dev).

1. ## [Transparency](https://cdn.jsdelivr.net/gh/asboy2035/Discord-Mods@main/Transparency.css)
   This theme provides transparency to the Discord UI. Also, be sure to check out Demureify and Semi-transparent main that were designed to improve this theme.
   > Note: You can get translucency on macOS with [Vencord](https://vencord.dev).

   ![Transparency](https://raw.githubusercontent.com/asboy2035/Discord-Mods/refs/heads/main/screenshots/transparency.png)

2. ## [Demureify](https://cdn.jsdelivr.net/gh/asboy2035/Discord-Mods@main/DemureifyNonTransparent.css)
   #### Designed for Transparency.
   Reduce the opacity of some non-transparent items for a more cohesive look with the Transparency theme.

   ![Demureified](https://raw.githubusercontent.com/asboy2035/Discord-Mods/refs/heads/main/screenshots/demureify.png)

3. ## [Beeg GIF Picker](https://cdn.jsdelivr.net/gh/asboy2035/Discord-Mods@main/Big-GIF-Emoji-Sticker-Picker.css)
   This theme makes your GIF/Emoji/Stickers picker take up most of the screen.

   ![Beeg GIF Picker](https://raw.githubusercontent.com/asboy2035/Discord-Mods/refs/heads/main/screenshots/beegGif.png)

4. ## [More Rounded Corners](https://cdn.jsdelivr.net/gh/asboy2035/Discord-Mods@main/MoreRoundedCorners.css)
   This theme gives some elements more rounded corners to look better (imo).

   ![Attachment](https://raw.githubusercontent.com/asboy2035/Discord-Mods/refs/heads/main/screenshots/roundedCorners/attachment.png)
   ![Context menu](https://raw.githubusercontent.com/asboy2035/Discord-Mods/refs/heads/main/screenshots/roundedCorners/context.png)
   ![Inbox](https://raw.githubusercontent.com/asboy2035/Discord-Mods/refs/heads/main/screenshots/roundedCorners/inbox.png)

5. ## [Blurred Headers](https://cdn.jsdelivr.net/gh/asboy2035/Discord-Mods@main/BlurredHeaders.css)
   Blurred Headers add a blur effect to headers in the Shop and Discovery.

   ![Blurred Headers](https://raw.githubusercontent.com/asboy2035/Discord-Mods/refs/heads/main/screenshots/blurredHeaders.png)

6. ## [Use System Fonts](https://cdn.jsdelivr.net/gh/asboy2035/Discord-Mods@main/SystemFonts.css)
   Use your platform's system font instead of Discord's.

   ![System Fonts](https://raw.githubusercontent.com/asboy2035/Discord-Mods/refs/heads/main/screenshots/systemFonts.png)

7. ## [Semi-Opaque Main](https://cdn.jsdelivr.net/gh/asboy2035/Discord-Mods@main/SemiOpaqueMain.css)
   #### Designed for Transparency.
   Makes the main content slightly darker to contrast the transparent outer level.

   ![Semi-Opaque Main](https://raw.githubusercontent.com/asboy2035/Discord-Mods/refs/heads/main/screenshots/semiOpaqueMain.png)
